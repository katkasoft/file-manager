# File Manager
A simple file manager written in tauri
## Features:
* **File navigation**
* **Native file execution**
* **Directory and file management**
* **Recursive deletion**
* **Renaming entities**
* **Copy/Paste via system clipboard**
* **Cut and Move functionality**
* **Recursive directory copying**
* **Built-in text file viewer**
* **Navigation history**
* **Hidden files toggling**
* **Multi-window support**
* **Keyboard shortcuts support**
* **Keyboard navigation**
* **Dynamic sorting**
